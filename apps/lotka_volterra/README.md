# LotkaVolterra

Lotka Volterra Equations

<https://en.wikipedia.org/wiki/Lotka%E2%80%93Volterra_equations>

## default values

With the default values the simulation will end up with this values:

Vegetation: 28'400

Herbivore:     570

Predator:       75
